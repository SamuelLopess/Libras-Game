# Libras Game

Libras game é um jogo da memória, onde o jogador deverá formar pares com o desenho do animal e sua respectiva representação em libras.

Desenvolvido para a plataforma Android, como trabalho de conclusão de curso, do Técnico de Informática (ETEC Americana - 2012)
